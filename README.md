# BurrowDB
BurrowDB is a fast, lightweight graph store built in Go for when your data is a tangled mess of connections. It’s designed to dig through complex relationships and deep links without breaking a sweat. If you're tired of forcing graph data into a square-shaped SQL hole, BurrowDB makes navigating those networks feel natural and snappy.
